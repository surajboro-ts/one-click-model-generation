# Strategy Overview

**Template system**
A set of clear templates (surfaces) that allows designers to create consistent interfaces with consistent structure and layouts

**Structured**

A clear set of the standardized interconnected part (widgets) elements to create the experience from. Widgets are comprised of multiple UI components. Widgets are defined by their reusable nature in various sections of the ThoughtSpot product. 

**Curation**

Remixing the widgets and surfaces that we have to make interfaces. Surface widgets are designed to be easily reconfigured for various interface needs.
