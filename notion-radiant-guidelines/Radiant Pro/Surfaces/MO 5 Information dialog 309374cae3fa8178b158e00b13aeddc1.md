### Examples

Ex.1 Show underlying data

**User:** As a user I wish to see the data set that is shown in my answer, so that I can double check the results. Therefore I go to the overflow menu and click on show underlying data.

**System response:** Opens a modal with a table that has data. 
Edit column opens another modal to edit the table.
Download CSV will download the CSV file to the user’s system.
Done will close the modal.

---

Ex.2 Query SQL

**User:** As a user I want to view the Query SQL behind my answer. So that I can better analyse my answer. Therefore, I go to the query details button on the right panel and click it. This opens a right rail. In the rail I go and click the Query SQL button.

**System response:** Opens a read only modal with the Query SQL.
Done button closes the modal.

---

Ex.3 Analysis details

**User:** As a user I want to view the Analysis Details of my SpotIQ analysis. This will help me understand all the drills and discoveries performed by SpotIQ. Therefore I go the overflow menu in spotIQ result screen of the answer and then click on Analysis Details.

**System response:** Opens a read only modal with Analysis Details of SpotIQ .
Done button closes the modal.

---

Ex.4 Invite teammate 

**User:** As a user I want to invite a teammate of mine to view an answer in TS. However I do not have the required permission to invite the member, Therefore when I click invite. 

**System response:** A modal opens letting me know that I do not have the required permission and I must contact the admin for the same. 
Done button closes the modal.

---

Ex.5 Delete

**User:** As a user I wanted to delete an object in TS so that I can import a copy of that object with updated data set. However when I try to delete, It does not delete as it isn’t the main object and that I must delete the main object to delete this.

**System response:** A modal opens letting me know that I cannot delete the object as depends on another object and I must delete that in order to delete this object. 
Clicking on link will take the user to the respected files.
Done button closes the modal.
