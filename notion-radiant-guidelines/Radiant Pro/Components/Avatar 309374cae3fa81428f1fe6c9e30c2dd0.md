# Avatar

## **Represents a user, or business, with a circular container including a picture or initials. The appealing visual nature of avatars makes ThoughtSpot more social.**

---

## **Types & examples**

Avatars may be used in three group styles: single, tray and grid. Additionally, any group style may use a choice of small, medium, or large avatars. /

**Single**

A standalone avatar represents one user, typically the current user or an object owner.

![Untitled](Avatar/Untitled.png)

A single avatar, in the context of an account dropdown menu.

**Tray**

A horizontal group of one to three avatars with negative margin that overlaps  to each other left-to-right.

![Untitled](Avatar/Untitled%201.png)

A Pinboard user’s avatar tray

**Grid**

Similar to a tray but presented in grids, useful for displaying large number of avatars in a group.

**Flat color thumbnails**

An assigned color with the first letter of the user name is displayed for avatars when an image is unspecified or unavailable (e.g. error when displaying the image).

Randomize the background colors of the avatars without a profile picture. Use the preset colors so the initials stand out visually.

![Untitled](Avatar/Untitled%205.png)

For accessibility, the contrast between the letter and the avatar background should be heigh. The system provides 6 default colors as followed.

| ✅ Do | ❌ Don’t |
| --- | --- |
| Separate avatar grids with section dividers. | Distort user profile pictures inside their containers. |
| On the far right, the first avatar try’s name label is displayed, along with a counter. | Leave empty space (white bezels) around cropped photos. |

## **Behavior**

Avatar can be presented in a clickable or non-clickable format.

A tooltip can be displayed with an avatar to be displayed on focus or on hover. The tooltip content should be set to the same value as the avatar label.

### **Specifications**

There are three size of avatar: Large (56px), Medium (32px), Small (24px).

For grids, the width of each avatar frame is 156px so that it can be evenly placed in a 720px modal dialog box.
