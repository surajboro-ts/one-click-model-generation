# Pagination

## It m**akes long, dense content easy to digest by dividing it into separate short pages.**

---

## **Types & examples**

**Range pagination**

Default pagination. Place on big space. Can go back to the first page.

![Untitled](Pagination/Untitled.png)

**Item pagination**

Place on small space. Only can go to the previous and next page, can't jump back to the first page. Showing the total result number is optional.

![Untitled](Pagination/Untitled%201.png)

---

## **Usage**

> Wrap the pagination number after 9,999, using the current number-wrapping logic. (Example: 8.15M.)
> 

> Show the total result number, if it makes sense in context. This step is optional.
>
