# Section alerts

### **Definition**

Section alerts inform the user that something that has happened in a specific section of the screen. They appear above the affected area and follow the same severity color code as the system alerts.

### **Usage**

Used to alert the user of something that has happened in a specific section of the screen. They appear above the affected area.

- Show local alerts in context to help capture the user’s attention.
- Align local alerts above the same module that caused the issue.
- Include a close button with a warning-style local alert.
- Never show multiple local alerts on the same page; use local muted alerts.
- Don’t include a close button with an error-style local alert.
