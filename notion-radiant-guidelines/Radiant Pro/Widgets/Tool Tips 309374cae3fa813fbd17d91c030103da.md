# Tool Tips

### **Definition**

Tooltips display descriptive text when users hover over, focus on, or tap an element.

Descriptive tooltip text either: describes what the element “is”  or completes messaging that otherwise could not be completed in the screen e.g truncated text.

### **Triggering the tooltips**

Tooltips are displayed on hover (with rare exceptions on click) with a **1 second delay** to prevent unwanted distraction to the main action. Moving the mouse out of the target object's hot zone dismisses the tooltip.

### **Content guidelines**

Use tooltips to add a small amount of information to page elements. Tooltips should only be used to help users understand the meaning or purpose of an icon or to show the full version of truncated text.

### **Writing tooltips**

- 1-2 word nouns to describe ‘what this is’ e.g “Filters”
- Not for calls to action (*Edit filters*)
- For icon only buttons a tool tip can provide a label
