# Select all checkbox

Number: 3
Content Rules: N/A
Default State: By default, this check box will be unselected.
Description: The select all check box allows the user to multi-select all the object items in the list. 
Dynamic Behavior: On clicking the check box all the checkboxes next to object items will be selected. 
Once all items have been selected, clicking again will unselect all items. 

If the user has selected one or more individual object select boxes then the select all check box will show a partially selected state. 
If the user clicks this state all the items that are currently selected will be unselected.
Error State: N/A
Object: OL.0 Object List
Type: Widget
