# Add values in bulk link

Number: 9
Content Rules: N/A
Default State: The add values in bulk link will be activated and blue in color.


Description: The add link lets users add multiple new value to the list. It does this by opening a new sub task modal with multi-line text input field.
Dynamic Behavior: On clicking the add values in bulk link, opens a new sub task modal with a multi-line text input widget.

Add value:
User can add a value(s) and select and apply it, these value(s) will be in the list. 

Remove value:
When the user visits the next time and they deselect these value(s), then this custom value(s) won't be there.
Error State: N/A
Object: ML.0 Multi-object list picker
