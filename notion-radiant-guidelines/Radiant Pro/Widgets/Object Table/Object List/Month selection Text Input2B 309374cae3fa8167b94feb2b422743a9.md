# Month selection: Text Input2B

Number: 7
Content Rules: N/A
Default State: This input will be empty by default, this is a required field.
Description: This input allows the user to set monthly frequency in context of dates of the month, IF the user has selected the “month” input from the primary frequency dropdown (2)
Dynamic Behavior: This input only appears when the user has selected “By date” option from the Month selection: Dropdown Input1 (5)
Values : Numbers between 1 to 31, separated by commas
Error State: The enter dates field requires at least one entry to validate this widget.
Object: FP.0 Frequency picker
