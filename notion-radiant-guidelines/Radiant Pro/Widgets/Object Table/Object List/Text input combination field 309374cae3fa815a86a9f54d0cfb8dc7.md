# Text input combination field

Number: 3
Content Rules: Empty field
Default State: Empty, unless data is filled then show data.
Description: Allows for longer, multi-line responses, for things like label descriptions.
Dynamic Behavior: By default, all the text input fields are required. When a text input field is optional, show “(optional)” in placeholder.
Error State: The field outline turn to red color
Object: IF.0 Input field
