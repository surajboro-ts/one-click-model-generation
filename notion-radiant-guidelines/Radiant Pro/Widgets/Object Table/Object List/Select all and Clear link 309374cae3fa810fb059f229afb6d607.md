# Select all and Clear link

Number: 8
Content Rules: N/A
Default State: The all and clear link will be activated and blue in color.
Description: It has options to select all objects and clear all selected objects in from the list.
Dynamic Behavior: On clicking the ‘Select all’ link, all the single object checkboxes next to object items will be selected. 
Header will read: column header (total number of objects)

On clicking the ‘clear’ link, all the single object checkboxes next to object items will deselect. 
Header will read: column name (0)

Select all and clear will select all/clear all the objects visible in the list at the given instance.
eg: If in search mode, only 4 options are shown, the select all will select all 4 options only.

Error State: N/A
Object: ML.0 Multi-object list picker
