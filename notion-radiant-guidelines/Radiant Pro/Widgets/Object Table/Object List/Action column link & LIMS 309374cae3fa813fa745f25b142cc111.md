# Action column link & LIMS

Number: 8
Content Rules: N/A
Default State: N/A
Description: The action column allows the user to apply actions to a specific row in the object list. The actions are always right aligned.

Actions include but not limited to: Favourite, Share, Edit, Duplicate, Export, Delete etc.
Dynamic Behavior: IF there are one or two actions then that will be represented as a blue link/s.

IF there are three or more actions then all the action links are placed in the elipse menu that is a drop down style menu, which gets activated once the user clicks over the menu icon.
Error State: N/A
Object: OL.0 Object List
Type: Widget
