# Number field

Number: 2
Default State: 1
Description: Text field that will always be a positive none 0 integer (natural number). The field should automatically validate itself
Dynamic Behavior: If the shortcut selected is anything apart from last quarter or last year this field will not be present.
Error State: This field will need to be validated for any non natural number and will automatically switch to nearest natural number and closest to any decimal entered or should prevent users from entering any decimal or negative values
