# Single object checkbox name

Number: 3
Content Rules: Maximum number of object items visible by default: 7
Default State: By default, the check box will be unselected.
Description: This check box allows the user to select an object next to it.
Dynamic Behavior: Clicking the check box selects the object, clicking again will unselect the object. Multiple single object checkboxes can be selected/unselected at the same time.

All the check box can also be selected/unselected if the user clicks the select all/ clear link at the top right of the object list respectively.
Error State: N/A
Object: ML.0 Multi-object list picker
Type: Widget
