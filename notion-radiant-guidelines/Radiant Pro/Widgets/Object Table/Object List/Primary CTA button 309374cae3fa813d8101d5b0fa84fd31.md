# Primary CTA button

Number: 6
Content Rules: The action word in the CTA should mirror the action in the title. 
e.g  Title: Sign in for access
CTA: Sign in
More on how to write great titles and CTAs for confirmation modals:
https://www.loom.com/share/f8b87cb2213c456baf363a49835bdfa5
Default State: -
Description: Clicking the primary action will complete the task and automatically close the modal dialog box.
Dynamic Behavior: The primary CTA should NEVER be disabled.
Object: MO.1 Simple dialog
Type: Surface Modal
