# Single object checkbox

Number: 5
Content Rules: N/A
Default State: By default, the check box will be unselected.
Description: This check box allows the user to select a row in the object list and apply an action to it.
Dynamic Behavior: Clicking the check box selects the row, clicking again will unselect the row. 

Multiple checkboxes can be selected at the same time.

The check box can also be unselected if the user clicks the select all check box at the top left of the object list.

All selection are effected on the current page, if users change to other pages, it won’t count 2 pages selections.
