Content Rules: Titles are typically nouns
Title should always make it clear which object list below
Title shoud be short with 1-2 word nouns e.g Liveboard, Tab, City etc.
Avoid overly long title
Don’t wrap or truncate title text
Title don’t need punctuation like periods or colons
Title should use sentence-case capitalization
