# Cancel/Back button

Number: 5
Default State: -
Description: Clicking the Cancel/Back button will close the current modal and return the user to its previous context. Cancel/Back undoes all applied changes.
Dynamic Behavior: In the main task dialog box, the CTA is “Cancel”
In the sub-task dialog box, the CTA is “Back”. Clicking the button, go back to the main task dialog box without submitting any data. 
When all the info are submited instantly or just check the info, the primary CTA is “Done”, and there is no need for secondary CTA.
Error State: -
Object: MO.2 Sub-task dialog
Type: Surface Modal
