# Search field

Number: 1
Content Rules: N/A
Default State: The search field would be empty
Description: The search field allows the user to search for specific objects in the object list.

Note: Search bar will be present in case of more than 20 options. Else it will not be present.

https://www.notion.so/309374cae3fa812895d1deffcef72709?pvs=21

Dynamic Behavior: Entering a search query will refresh the object list with objects that match the search query term. 
The object list will refresh dynamically as the user enters their search query.
Highlight the search term in the results.
Error State: If there are no object results for a search query, the object list should show an empty state graphic.
Object: SOL.0 Single object list picker
