# Object Search

Number: 2
Default State: The default state is to just show an icon of the object search
Description: The default state is to just show an icon of the object search
Dynamic Behavior: Clicking on the icons will open the object search field. The user can then enter a text string into the field and press return or click the the icons to the far right of the field to iniate an object search.
Initiating a search will refresh the page and show the object search results page.
Error State: If the user does not enter a string in object search field then the page and show the object search results page with a zero results view.
