# Month selection: Dropdown Input2A

Number: 6
Content Rules: N/A
Default State: By default “First” + “Monday” is selected.
Description: This input allows the user to set monthly frequency in context of days of the week, If the user has selected the “month” input from the primary frequency dropdown (2)
Dynamic Behavior: This input only appears when the user has selected “On the” option from the Month selection: Dropdown Input1 (5)
Values : 
Dropdown 1: “First”, “Second”, “Third”, “Fourth”, “Last”
Dropdown 2: “Monday”, “Tuesday”, “Wednesday”, “Thursday”, “Friday”, “Saturday”, “Sunday”, “Day”
Error State: N/A
