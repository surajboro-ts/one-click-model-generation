# Time period field

Number: 3
Default State: Quarter(s)
Description: Dropdown field that will have multiple values to choose the time period
Dynamic Behavior: Can have any of the below values
1.Second(s)s
2.Min
3.Hour(s)
4.Day
5.Week
6.Month
7.Quarter
8.Year(s)
Error State: NA - It will always have a value
Object: RDF.0 Date and time picker
