# Primary CTA button

Number: 6
Default State: -
Description: Clicking the primary action will complete the task.
Dynamic Behavior: In the main task dialog box, the CTA is the action for main task. Click the button complete the task and close the dialog box.
In the sub-task dialog box, the CTA is the action for sub-task. Click the button complete the task and go back to the main task dialog box.
The primary CTA should NEVER be disabled.
Error State: -
Object: MO.2 Sub-task dialog
Type: Surface Modal
