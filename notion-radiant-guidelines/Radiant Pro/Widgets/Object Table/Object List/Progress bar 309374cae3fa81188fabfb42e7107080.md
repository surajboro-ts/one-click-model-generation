# Progress bar

Number: 4
Content Rules: N/A
Default State: The bar will be shown in step 1 phase.
Description: This is used in wizard modal to show how much progress has been made in a particular process.
Dynamic Behavior: Increases in length with completion of every step in proportion to the number of step.

eg: If a wizard is of 3 steps then the bar will be divided in 3 equal parts. Each part will highlighted with blue color when the step is completed.
Error State: N/A
