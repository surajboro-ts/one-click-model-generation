# Progress bar

Number: 4
Content Rules: -
Default State: The background is always Grey20. 

The stepper has two states. First stepper will always be Blue60 and the rest will be Grey30.
Description: Shows which step the current task is on.

The progress bar determine the percentage of the completed task.

It contains a background and stepper bars. There will always be more than one stepper bars. 
Dynamic Behavior: On moving to the next step, the stepper in the progress bar changes from  Grey30 to Blue60.

The height of the progress bar is 4 px

Based on the number of steps in wizard , the progress bar will have steppers. The space between the steppers is 4 px.
Error State: -
Object: MO.3 Wizard dialog
Type: Surface Modal
