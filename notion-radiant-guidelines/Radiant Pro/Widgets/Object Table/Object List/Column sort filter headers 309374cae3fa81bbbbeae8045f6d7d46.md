# Column sort/filter headers

Number: 4
Content Rules: N/A
Default State: The default state is determined by the needs of the project.
Description: Column sort headers allow users to reorder an object list.

These are accessed by clicking on the column header.

The headers also has a grey 20 hover state. On hover the cursor changes to a hand to denote click-ability.

https://www.loom.com/share/15ebd8bdb3084863961121ca8f337fd8?sid=aa057ef6-e22d-4bc7-b386-9ba430421402 for hover state.
Dynamic Behavior: Clicking on the column header will open the menu, users can choose to filter or sort.

Clicking on the Filter will open the filter modal dialog box.

Clicking on the Sort will resort to the object list according to the column type. I.e chronologically, alphabetically, low to high. 

Once clicked the Sort ascending or Sort descending will display an up or down arrow to denote both that this is the column that the object list is sorted by and also the direction of the sort order.
