## DP**.0 Date picker**

**Overview** 
The date picker widget allows the user to select a specific fixed date. 

**User Action**: Users want to select particular dates to filter.

**System Response**: The system starts in a default state showing a condition dropdown displaying- between along with two date field for start and end date. Users can change conditions from the dropdown and the rest of the fields change along with it.
