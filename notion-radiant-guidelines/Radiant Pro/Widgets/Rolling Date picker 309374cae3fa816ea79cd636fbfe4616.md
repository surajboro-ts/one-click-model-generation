## RDF**.0 Rolling data filter**

**Overview** 
OVERVIEW TEXT 

**User Action**: Users want to select a date to filter with respect to the current date (anchor date).

**System Response**: The system starts in a default state showing a condition dropdown displaying- last, number filed- 1, and time period field -Quarter. Users can change conditions from the dropdown, and the rest of the fields change along with it.
